# Hospital Operations & Patient Flow Performance Dashboard

## Project Overview
This Power BI project analyzes hospital operations, patient flow, doctor performance, and revenue trends using interactive dashboards.

## Tools Used
- Power BI
- Power Query
- DAX
- Row-Level Security (RLS)

## Features
- Appointment analysis
- Revenue tracking
- Doctor performance KPIs
- Patient flow monitoring
- Dynamic filtering
- Role-Based Access Control (RLS)

## Dashboard Pages
1. Executive Overview
2. Patient Flow Analysis
3. Doctor & Revenue Performance

## Key KPIs
- Total Revenue
- Total Patients
- Total Appointments
- Total Doctors
- Revenue per Doctor
- Treatment Trends

## Data Modeling
- Star Schema
- Fact & Dimension tables
- DAX measures
